# Welcome 👋

### My name is Ye(烨) Han(韩). 

- 🔭 I am a Ph.D. candidate in [Vehicle Engineering](https://auto.tongji.edu.cn/) at [Tongji University](https://www.tongji.edu.cn/) under the supervision of [Prof. Lijun Zhang](https://auto.tongji.edu.cn/info/1180/6453.htm).     - 🛣️ Multi-Vehicle Cooperative Decision Making
